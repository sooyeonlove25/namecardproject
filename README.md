# namecardproject
# 명함 사용자로부터 이름을 input 입력 1단계 정리글
명함 입력 시작
"명함입력" 메시지를 출력한다.
이름 입력
put으로 입력받고 공백을 제거한 후 name_tmp 변수에 저장한다.
strip 을 쓰는 이유는, 일관된 변수를 저장하기 위해서다
ex) "홍길동"과 " 홍길동"을 다르게 저장하면 검색이나 비교 시 문제가 생긴다.
전화번호 및 소속 입력
전화번호와 소속을 리스트에 바로 입력받아 각각 phones 및 companies 리스트에 추가한다.
명함 정보 저장
name_tmp임시 변수에 입력받은 이름을 append메서드를 통해 names 리스트에 추가한다.
명함 추가 완료 메시지 출력
{이름}님의 명함이 추가되었습니다. 메시지를 통해 정보를 출력한다.
